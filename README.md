An example of creating custom validation rules for terraform.

## Prerequisites

* [Install Terraform](https://learn.hashicorp.com/terraform/getting-started/install)

## Usage

### Validation Fail

* Run terraform

        $ terraform init
        $ terraform plan

### Validation Pass

* Uncomment variables in terraform.tfvars

* Run terraform

        $ terraform init
        $ terraform plan
